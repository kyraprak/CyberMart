# CyberMart 🛒⚡

CyberMart is a Java-based backend project that simulates a futuristic online store for next-generation tech products. It models core e-commerce functionality such as product listings, shopping cart management, and checkout calculations.

This project is designed to demonstrate object-oriented programming (OOP) principles in Java, including encapsulation, class design, and collection handling.

---

## 🚀 Features

- Product catalog system (innovative/futuristic tech items)
- Add and remove products from a shopping cart
- View cart contents
- Calculate total price at checkout
- Count items in cart
- Read-only access to cart items for safety
- Clean OOP structure (Product, Cart, Main)

---

## 🧠 Tech Stack

- Java (JDK 17+ recommended)
- IntelliJ IDEA (development environment)
- Standard Java Collections (`ArrayList`, `List`)
